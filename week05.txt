
1교시
다른사람의 원격리포지토리를 클론 후 새파일 추가 또는 기존파일 편집 후 로컬리포지토리에 커밋
깃 명령어
git clone 원격리포지토리주소
git clone 원격리포지토리주소 .
git config --list 
git remote --list

리눅스 명령어
vi편집기 화면이 나타나면 j(아래), k(위), q(편집기 탈출)

2교시
원저작자의 바인딩된 원격리포지토리 주소를 내 원격리포지토리 주소로 업데이트

확인 git remote -v
삭제 git remote remove origin
등록 git remote add origin 주소

git push
->
git push -u origin main
혹은
git push --set-upstream origin main
