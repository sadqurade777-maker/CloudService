3교시
-사용자 등록
git config --global user.email "사용자 이메일 주소" (깃 사용자 이메일 주소 등록)
git config --global user.name "사용자 필명" (깃 사용자 아이디 등록)

- 깃 명령어
git add 파일명 (지정한 파일을 스테이지에 올림, ㅅㅈ찍음)
git add . (현재 폴더의 모든 폴더를 스테이지에 올림)
. (현재 모든 폴더)
.. (상위 모든 폴더)
git commit 파일명 -m "커밋 메세지" (스테이지에 올린 파일들을 commit 기록함. 해시코드가 부여되고 커밋한 아이디/이메일/커밋시간을 남김)

-이전으로 돌아가기
git log (커밋 기록 보기 명령(타임머신), 해시코드/아이디/이메일/커밋시간/커밋이메일)
git checkout 해시코드 (특정 시점으로 이동)
git checkout - (바로 이전으로 이동)
